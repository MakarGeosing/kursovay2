# Система бронирования железнодорожных билетов

## Описание
Десктопное приложение для автоматизации процессов бронирования и продажи железнодорожных билетов. Разработано для использования в кассах РЖД.

## Функциональность
- Аутентификация пользователей (администратор, пользователь)
- Визуальный выбор мест в вагоне
- Бронирование билетов
- Управление бронированиями
- Просмотр отчетов и статистики

## Технологии
- Python 3.9+
- PyQt5 - графический интерфейс
- MySQL 8.0 - база данных
- MySQL Workbench - администрирование БД
